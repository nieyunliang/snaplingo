# Snaplingo 功能文档

## 1. 产品概述

Snaplingo 是一款原生 macOS 菜单栏截图翻译工具。它面向阅读外语内容的用户，提供从截图、OCR 识别、原位翻译到复制的一站式流程。

核心链路：

```text
菜单栏或快捷键唤起 -> 拖拽区域或点击窗口 -> 原位浮层展示 -> 基础标注或图内翻译 -> 复制
```

## 2. 目标用户

- 开发者：阅读英文文档、报错信息、GitHub issue、技术博客。
- 学生和研究者：阅读论文、PDF、课程材料。
- 产品和设计人员：分析海外产品界面、竞品截图、设计说明。
- 办公用户：处理邮件、网页、图片中的外语文本。

## 3. 运行环境

- 平台：macOS 14 及以上。
- 应用形态：Swift Package 构建的原生 macOS 菜单栏应用。
- 主要系统能力：AppKit、SwiftUI、ScreenCaptureKit、Vision、SMAppService。
- 必要权限：屏幕录制权限。未授权时，应用会提示用户请求权限或跳转系统设置。

## 4. 用户入口

### 4.1 菜单栏入口

应用启动后常驻 macOS 菜单栏，菜单提供以下入口：

- 截图：进入统一截图模式。拖拽鼠标截取区域，单击高亮窗口截取对应窗口。
- 设置：打开配置窗口。
- 退出：关闭应用。

### 4.2 全局快捷键

- 默认截图快捷键：`Option + A`。

## 5. 核心功能

### 5.1 统一截图

用户通过菜单或 `Option + A` 进入统一截图模式。

功能点：

- 显示半透明截图遮罩。
- 鼠标悬停时高亮可截取窗口。
- 拖拽鼠标并松手后立即截取当前区域，并进入原位冻结编辑浮层。
- 区域截图完成后统一使用冻结图片旁的原位工具条进行标注、翻译、复制、保存、完成或关闭。
- 单击高亮窗口后立即截取窗口。
- 支持鼠标放大镜辅助定位。
- 支持 `Esc` 取消截图。
- 区域截图限定在单个显示器内，窗口截图支持任意显示器。
- 支持 Retina 缩放和多显示器坐标转换。

### 5.2 原位截图工具条

区域截图在松开鼠标后立即完成。截图确认后，冻结图片显示在原屏幕位置，操作栏优先显示在截图区域下方。

操作功能：

- 箭头：在图片上连续绘制多个箭头。
- 矩形：在图片上绘制矩形边框。
- 圆形：在图片上绘制圆形边框。
- 撤销：移除最近绘制的标注。
- 翻译：识别图片中的外语，并切换显示或隐藏覆盖在原文字位置的中文译文。
- 复制：复制当前合成图片。
- 完成：复制当前合成图片并关闭浮层。
- 关闭：关闭截图浮层，不写入截图文件。

## 6. OCR 文字识别

Snaplingo 使用 macOS Vision 框架执行本地 OCR。

功能点：

- 使用 `VNRecognizeTextRequest` 进行本地识别。
- 默认先使用快速识别；未识别到文字或平均置信度较低时，自动回退到准确识别。
- 支持用户配置 OCR 语言列表。
- 默认语言为 `en-US` 和 `zh-Hans`。
- 准确识别阶段支持语言纠错。
- OCR 结果包含全文、识别语言、平均置信度和文字块坐标。
- 文字块坐标用于图内中文替换。
- 未识别到文字时，原位工具条会展示错误状态，但截图仍可复制。

## 7. 翻译功能

### 7.1 图内翻译

截图完成后，用户可以从原位工具条点击“翻译”。

功能点：

- 截图工具条固定将外语翻译为中文。
- OCR 和翻译只在用户点击“翻译”后触发。
- 翻译打开后再次点击会隐藏译文；继续点击会恢复显示已缓存译文，不重复调用翻译接口。
- 按 OCR 文字块批量翻译，并将中文覆盖到原文字位置。
- 同一截图中内容完全相同的文字块只请求翻译一次，再覆盖到各自位置。
- 纯数字、纯符号和已识别为中文的文字块不会重复翻译。
- OCR 结果为空时不会触发翻译。
- 翻译失败时在原位工具条展示错误状态，用户可重试。

### 7.2 翻译服务

应用支持三类翻译提供方：

- DeepSeek：调用 DeepSeek Chat Completions 接口。
- 离线词典：使用内置词典和术语表进行基础替换。
- 翻译记忆：原位翻译会按 OCR 文字块优先命中本地已有翻译结果；未命中的文字块会批量路由到 DeepSeek。

DeepSeek 配置项：

- DeepSeek API Key：保存到本地应用偏好设置。
- DeepSeek 模型：默认 `deepseek-v4-flash`。
- DeepSeek 接口地址：默认 `https://api.deepseek.com/chat/completions`。
- 网络超时：30 秒。

### 7.3 翻译风格

支持四种翻译风格：

- 自然：译文自然、流畅，并保留原意。
- 直译：尽量保留原文句式和信息顺序。
- 专业：使用适合技术、学术和商务场景的专业表达。
- 简洁：压缩表达，只保留核心意思。

### 7.4 术语表

用户可以在设置中维护术语表。

格式：

```text
source=target
source=>target
source:target
source：target
```

使用方式：

- DeepSeek 翻译时，术语表会作为系统提示的一部分注入。
- 离线翻译时，术语表会优先于内置词典进行替换。

### 7.5 翻译记忆

翻译记忆会记录已完成的翻译结果，后续相同原文和翻译风格会直接命中对应的中文结果。

功能点：

- 默认启用。
- 匹配时会归一化空白和大小写。
- 命中后结果 provider 标记为翻译记忆。
- 存储文件为应用支持目录下的 `translation-memory.json`。

## 8. 原位截图浮层

截图完成后，应用会在原屏幕位置展示冻结图片，并在附近展示轻量工具条。

特性：

- 图片和工具条均使用置顶 `NSPanel`。
- 工具条优先位于图片下方，空间不足时移动到上方。
- 基础标注与图内翻译使用非破坏式编辑状态，复制时才合成最终图片。
- 点击“完成”会将合成后的最终图片复制到系统剪贴板，并关闭浮层。
- 关闭浮层不会产生截图文件。

## 9. 基础标注

截图浮层支持连续绘制箭头、矩形和圆形，并可逐步撤销最近一次标注操作。

## 10. 设置项

### 10.1 快捷键

- 统一截图默认快捷键展示：`Option + A`。

### 10.2 识别与翻译

- 翻译风格。
- OCR 语言列表。
- 启用翻译记忆。
- 术语表。

### 10.3 DeepSeek

- 翻译服务。
- DeepSeek 模型。
- DeepSeek 接口地址。
- DeepSeek API Key。

### 10.4 通用

- 窗口截图包含阴影。
- 开机启动。

### 10.5 权限

- 屏幕录制权限状态。
- 请求屏幕录制权限。
- 打开系统权限设置。

## 11. 数据与隐私

本地存储：

- 翻译记忆 JSON。
- 用户设置。
- 本地应用偏好设置中的 DeepSeek API Key。

仍可能发生的外部请求：

- 当翻译服务选择 DeepSeek 或翻译记忆未命中并回退 DeepSeek 时，OCR 原文会发送到用户配置的 DeepSeek 接口。
- 当选择离线词典并关闭其他在线服务时，翻译过程不依赖网络。

## 12. 异常与提示

常见异常：

- 缺少屏幕录制权限。
- 截图失败。
- 未找到可截取窗口。
- 图片转换失败。
- 未识别到文字。
- 未配置 DeepSeek API Key。
- 翻译响应无法解析。

处理方式：

- 权限异常会引导用户请求权限或打开系统设置。
- OCR 和翻译异常会显示在原位工具条状态区域。
- 用户可重新点击翻译按钮重试。

## 13. 已知限制

- 区域截图暂不支持跨显示器拖拽。
- iCloud 同步尚未实现；完整同步还需要 entitlements、签名和冲突处理。
- Screen Recording 权限无法在无头测试环境中自动授予，因此真实截图烟测仍需人工或权限门控。
- 离线翻译目前是术语表和小型内置词典替换，不能替代完整机器翻译。

## 14. 验收建议

基础截图链路：

- 使用 `Option + A` 进入截图模式，拖拽区域并松手，验证松手后立即进入原位冻结编辑浮层。
- 在原位工具条点击箭头、矩形或圆形，验证可在冻结截图上绘制对应标注。
- 在原位工具条点击翻译，验证可开始图内翻译；点击完成，验证合成图片被复制并关闭浮层。
- 单击高亮窗口，验证窗口截图。
- 验证原位截图、箭头、矩形、圆形、撤销、图内翻译、复制、完成和关闭。
- 验证复制或关闭截图浮层不会产生截图文件。

截图模式：

- 验证窗口截图，分别测试包含和不包含阴影。
- 分别在主显示器和副显示器执行区域截图与窗口截图。

识别与翻译：

- 验证截图翻译只在点击翻译按钮后触发，并验证再次点击隐藏译文、继续点击恢复缓存译文且不重复请求接口。
- 测试缺少 DeepSeek API Key 时的错误提示。
- 测试离线词典翻译。
- 测试术语表优先替换。
- 重复翻译相同内容，验证翻译记忆命中。

## 15. 后续规划

建议优先级：

- 按需恢复长截图：接入 Accessibility，识别实际滚动容器，做智能拼接和重叠去重。
- 完整 iCloud 同步：增加 entitlements、容器读写、冲突合并和状态反馈。
- 扩展翻译服务：支持 DeepL、Google Translate、本地模型等 provider。
- 增强 OCR 语言配置：提供可视化语言选择，而不是手写语言代码。
- 增加表格识别：支持复杂表格结构和导出 CSV。
- 增加权限异常 UI 测试和真实截图烟测。
